# Ruta C

Estado: `planned`

## Propósito

Ruta C es la primera ruta curricular completa de Lumen. Debe enseñar programación en C mediante una progresión guiada, práctica, medible y acumulativa.

## Principio

La ruta no es una colección ordenada de archivos. Cada módulo introduce, practica, combina y evalúa conocimientos.

## Módulos iniciales

1. Fundamentos y entrada/salida.
2. Variables, tipos y operadores.
3. Condicionales.
4. Bucles.
5. Funciones.
6. Arrays.
7. Cadenas.
8. Matrices.
9. Punteros.
10. Structs.
11. Archivos.
12. Memoria dinámica.
13. Integración final.

Los nombres y subdivisiones pueden evolucionar, pero las dependencias pedagógicas deben mantenerse.

## Estructura de cada módulo

- introducción breve;
- ejercicios básicos;
- ejercicios principales;
- práctica de errores comunes;
- actividad combinada;
- quiz conceptual;
- reto de dominio;
- proyecto cuando corresponda.

## Progresión

El usuario no necesita completar cada ejercicio existente. Cada módulo distingue:

- obligatorios;
- recomendados;
- refuerzo;
- extra;
- gate;
- proyecto.

El siguiente módulo se desbloquea cuando se cumplen las reglas del gate. El número de ejercicios completados por sí solo no basta.

## Proyectos de referencia

- Blackjack.
- Bingo.
- Memory Game.
- Sistemas de menús.
- Mini CRUD con structs y archivos.
- Proyecto final con múltiples conceptos.

## Resultado esperado

Al completar Ruta C, el estudiante debe poder leer, diseñar, depurar y construir programas de C sin depender de copiar soluciones. También debe entender errores del compilador, descomponer problemas y usar el editor con fluidez.
